<Section icon={Shield} title="Controller Layout and DualSense Setup">

The combat system requires precise execution of combos, evasive maneuvers, and quick-time events. Customizing the controller layout can significantly improve reaction times. The default scheme assigns primary attacks to the face buttons, but some players prefer mapping dodge or skill triggers to the bumpers for better camera control during combat.

The DualSense controller provides immersive haptic feedback that corresponds to the weight of different weapons. Heavy broadswords produce strong, slow vibrations, while swift daggers offer light, rapid pulses.

| Action | Default Button | Recommended Alternative | Purpose of Change |
| :--- | :--- | :--- | :--- |
| **Basic Attack** | Square | Square | Standard combo execution |
| **Evasion / Dodge** | Circle | R1 | Keep thumb on right stick |
| **Hunter Skill 1** | Triangle | Triangle | Quick access to primary skill |
| **Hunter Skill 2** | Cross | Cross | Quick access to secondary skill |
| **Extreme Evasion** | R1 | L1 | Easier timing for counterattacks |

  <Step num="1" title="Access Settings Menu">
    Navigate to the main menu, select the gear icon, and open the Controller submenu.
  </Step>
  <Step num="2" title="Select Custom Preset">
    Switch the control scheme from Default to Custom to unlock individual button mapping.
  </Step>
  <Step num="3" title="Rebind Evasion Controls">

  </Step>
  <Step num="4" title="Adjust Trigger Sensitivity">

  </Step>

<Section icon={Target} title="Troubleshooting and System Maintenance">

Even with console optimization, players may occasionally encounter performance hiccups, network disconnections, or visual glitches. Keeping the game client updated and maintaining adequate storage space helps prevent common errors. Because the game requires an active internet connection for progression tracking, a stable network setup is highly recommended.

If you experience persistent frame rate drops, clearing the console system cache or reinstalling the game data can resolve underlying file corruption issues.

| Issue | Potential Cause | Recommended Solution |
| :--- | :--- | :--- |
| **Stuttering in lobbies** | High player density | Switch to a less populated channel |
| **Connection errors** | Unstable wireless signal | Use a wired LAN connection if possible |
| **Texture pop-in** | External drive bottleneck | Move the game to the internal console SSD |
| **HDR washed out** | Incorrect system calibration | Adjust HDR settings in the system dashboard |

<Callout type="warning" title="Storage Space Requirement">
Ensure you have at least 10 GB of free space beyond the initial installation size to accommodate temporary patch files and system updates.
</Callout>
