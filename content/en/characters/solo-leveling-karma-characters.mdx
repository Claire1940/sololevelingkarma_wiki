<Section icon={Target} title="Character Progression through Blessings">

In Solo Leveling: KARMA, character power is transient. While Jinwoo has permanent base stats, his true potential in a run is dictated by the **Blessing System**. This system ensures that no two runs feel the same, even if you are using the same set of **solo leveling karma characters**.

Blessings are categorized into several types, each affecting different aspects of your character:

| Blessing Type | Effect on Characters | Example Synergies |
| :--- | :--- | :--- |
| **Skill Enhancement** | Increases ability level/damage | Greatly buffs Igris's ultimate |
| **Stat Growth** | Permanent (per run) HP/ATK boosts | Essential for surviving Vulcan |
| **Weapon Synergy** | Adds effects to specific weapons | Bleed damage for Daggers |
| **Shadow Synergy** | Buffs the Shadow Army's AI/Stats | Increases Iron's taunt duration |
