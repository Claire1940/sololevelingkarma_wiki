<Section icon={Map} title="俺だけレベルアップな件：KARMAのカルマシステムを理解する">

この**「俺だけレベルアップな件：KARMA」カルマシステム攻略ガイド**は、次元の裂け目の中で繰り広げられる27年間の戦争を生き抜くための基礎知識となります。本作のようなローグライト・アクションRPGにおいて、「カルマ」は君主たちに対する水篠旬（ソン・ジヌ）の終わりなき闘争の輪廻を表しています。従来の固定されたRPGとは異なり、本作の成長システムは「周回（ラン）ベース」です。つまり、ダンジョンに挑むたびに、戦闘スタイルやアビリティの組み合わせを新しく洗練させる機会が与えられます。

このシステムは、プレイヤーの試行錯誤を促すように設計されています。戦闘ステージを深く進むにつれて、その周回限定であなたの「カルマ（運命）」を決定づける一時的なパワーアップ（祝福）を獲得できます。選択した武器、手に入れた祝福、そして使役する影の兵士たちの相互関係を理解することが、激化する魔法獣の猛攻を生き延びるための鍵となります。

祝福システムは、この**「俺だけレベルアップな件：KARMA」カルマシステム攻略ガイド**の中で最もダイナミックな要素です。祝福は単なるステータス上昇にとどまらず、1回の周回におけるキャラクターのビルドを構築する重要なパーツとなります。ネットマーブル（Netmarble）は、ドロップのランダム性（RNG）に応じてプレイヤーが戦略を適応させられるよう、毎回異なるプレイ体験が楽しめる設計を施しています。

祝福は、戦闘効率への影響度によって分類されます。アグレッシブな近接戦闘に向いたものもあれば、戦術的な遠距離戦闘に適したものもあります。カルマの周回を成功させる秘訣は、現在の武器装備と影の軍団の構成にどの祝福が最もマッチしているかを素早く見極めることです。

| 祝福タイプ | 主なメリット | 推奨武器 |
| :--- | :--- | :--- |
| **攻撃の奔流（Offensive Surge）** | 物理ダメージの大幅な増加 | 双剣 / 大剣 |
| **魔力の流れ（Mana Flow）** | クールダウン短縮とMP回復量アップ | 特殊魔法武器 |
| **影の共鳴（Shadow Resonance）** | 影の兵士の攻撃力と持続時間の向上 | 全武器（影特化ビルド） |
| **回避機動（Evasive Maneuver）** | 回避成功時に一時的な無敵時間を付与 | 短剣 / 弓 |
| **要塞の意志（Fortress Will）** | ボスから受けるダメージを大幅に軽減 | ハンマー / 盾 |

<Callout type="success" title="シナジー効果について">
「魔力の流れ」と「影の共鳴」を組み合わせることで、影の軍団をほぼ常時召喚した状態を維持できます。これは2026年現在、敵が密集する高密度フロアをクリアする最も効果的な方法です。
</Callout>

影の軍団への深い理解なしに、この**「俺だけレベルアップな件：KARMA」カルマシステム攻略ガイド**は完成しません。水篠旬としての最大の強みは、倒した敵を忠実な兵士として蘇らせる能力です。『俺だけレベルアップな件：KARMA』において、影の軍団はプレイヤー自身のスキルと敵の圧倒的な数とのギャップを埋める強力なサポートシステムとして機能します。

<Steps>
  <Step num="1" title="影の総官たちを解放する">

  </Step>

  </Step>
  <Step num="3" title="コンボ攻撃を発動する">

  </Step>
  <Step num="4" title="影の特性を強化する">
    周回外で獲得できる永続的な育成リソースを使用して影の軍団の基本ステータスを強化し、高難易度ティアでも十分に戦える状態を維持しましょう。

<Section icon={Swords} title="武器切り替えと戦闘メカニズム">

『俺だけレベルアップな件：KARMA』の戦闘はテンポが速く、常に状況に合わせた適応が求められます。本**カルマシステム攻略ガイド**で強調したいのが「武器切り替え（ウェポンスワップ）」メカニズムです。旬は複数の武器を装備し、コンボの途中でそれらを瞬時に切り替えることができます。これは見た目の派手さだけでなく、敵の持つ特定の耐性や弱点に対応するために必要不可欠なシステムです。

| 武器カテゴリー | 戦闘での役割 | 有効な敵 |
| :--- | :--- | :--- |
| **短剣 / 双剣** | 素早い近接DPS | 小回りの利く俊敏な魔法獣 |
| **大剣 / ハンマー** | 重い一撃と衝撃ダメージ | 盾持ちの敵や動きの遅いボス |
| **弓 / 遠距離武器** | 距離のコントロール | 飛行する敵やフィールドのギミック |
| **大鎌 / 特殊武器** | 広範囲の雑魚処理 | 大群で押し寄せる下級モンスター |

効果的な戦闘を行うには、これらの武器をローテーションさせて絶え間なく圧力をかける必要があります。例えば、大剣でボスの体勢（ポスチャー）を崩し（ブレイク）、無防備になった瞬間に短剣に切り替えてクリティカルヒットの嵐を浴びせる、といった連携が強力です。

<Callout type="info" title="ビルドの多様性">
獲得する「カルマ」の祝福は、特定の武器タイプを強化するものが多いことを忘れないでください。もし「弓の熟練度」の祝福を多く獲得した場合は、手持ちの短剣のレベルが高くても、その周回では遠距離プレイスタイルに特化した方が効率的です。
</Callout>

</Section>
