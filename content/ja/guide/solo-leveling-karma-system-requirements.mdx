<Section icon={Shield} title="俺だけレベルアップな件：KARMA システム要件：PCスペック">

PCでプレイすることで、強化されたシャドウ解像度、グローバルイルミネーション、無制限のフレームレートなど、最も視覚的に素晴らしい体験が可能になります。最高のパフォーマンスを目指すプレイヤーは、以下のハードウェア階層を確認し、自身のPCがアクションに備えられているかチェックしてください。

最低スペックは低設定で1080p解像度・30 FPSを出すことを目標としており、推奨スペックは高設定で1080p・60 FPSを出すことをターゲットとしています。

| 構成要素 | 最低スペック | 推奨スペック |
|---|---|---|
| OS | Windows 10 (64-bit) | Windows 10 / 11 (64-bit) |
| CPU | Intel Core i5-8400 / AMD Ryzen 5 1600 | Intel Core i7-9700K / AMD Ryzen 7 3700X |
| RAM | **8 GB RAM** | **16 GB RAM** |
| GPU | NVIDIA GTX 1060 6GB / AMD RX 580 | NVIDIA RTX 2060 / AMD RX 5600 XT |
| DirectX | Version 11 | Version 12 |
| ストレージ | 50 GB 以上の空き容量 (HDD) | 50 GB 以上の空き容量 (SSD) |

4K解像度、最高設定、レイトレーシング有効でゲームを楽しみたい熱心なプレイヤーには、エリートティアの構成が必要です。このセットアップでは、NVIDIA DLSSやAMD FSRなどの高度なアップスケーリング技術を活用し、激しいボス戦の間も安定したフレームレートを維持します。

| 構成要素 | ウルトラ4Kスペック |
|---|---|
| OS | Windows 11 (64-bit) |
| CPU | Intel Core i7-12700K / AMD Ryzen 7 5800X3D |
| RAM | **32 GB RAM** |
| GPU | NVIDIA RTX 4070 / AMD RX 7800 XT |
| DirectX | Version 12 |
| ストレージ | 50 GB 以上の空き容量 (NVMe SSD) |

<Callout type="warning" title="ストレージに関する注意">
標準的なハードディスクドライブ（HDD）にゲームをインストールすると、音声のズレ、テクスチャの読み込み遅延、ロード画面の長時間化が発生する可能性があります。
</Callout>

<Section icon={Swords} title="モバイルシステム要件とパフォーマンス">

Netmarbleは、幅広いデバイスで動作するようにモバイル版を最適化しています。しかし、戦闘システムのスピードが速いため、混雑したボス戦でのサーマルスロットリング（熱による性能低下）や突然のフレームドロップを避けるには、推奨スペックを満たすデバイスを使用することが極めて重要です。

| プラットフォーム | 最低スペック | 推奨スペック |
|---|---|---|
| Android OS | Android 9.0 以上 | Android 11.0 以上 |
| Android SoC | Snapdragon 845 / Exynos 9810 | Snapdragon 8 Gen 1 / Dimensity 9000 |
| Android RAM | **4 GB RAM** | **8 GB RAM** |
| iOS OS | iOS 14 以上 | iOS 16 以上 |
| iOS デバイス | iPhone XS / iPad Air 3 | iPhone 14 Pro / iPad Pro (M1) |
| iOS RAM | **4 GB RAM** | **6 GB RAM** |

特定のデバイスでどのようなパフォーマンスが得られるかを判断しやすくするため、モバイルデバイスを3つのパフォーマンス階層に分類しました。これにより、グラフィックの忠実度やバッテリー持ちの目安を確認できます。

  <Card title="低価格帯デバイス">

    - **推奨**: ブルームとモーションブラーを無効にする
  </Card>
  <Card title="ミドルレンジデバイス">
    - **目標**: 中設定で 45-60 FPS
    - **欠点**: 長時間のセッションで中程度の発熱
    - **推奨**: 安定性のためにフレームレートを 45 FPS に制限する

長時間のプレイセッション中は、スマホクーラーを使用したり、厚い保護ケースを取り外したりすることで、サーマルスロットリングを防ぎ、ピークパフォーマンスを維持できます。

<Section icon={Sparkles} title="ステップバイステップ最適化ガイド">

システムが最低要件をかろうじて満たしている場合でも、いくつかのソフトウェア最適化を行うことで、PCから余分なパフォーマンスを引き出すことができます。以下の手順に従って、ゲームの準備を整えてください。

<Steps>
  <Step num="1" title="グラフィックスドライバーの更新">
    NVIDIA、AMD、またはIntelから直接最新のドライバーをダウンロードしてください。GPUメーカーは、Netmarbleの新作タイトルに合わせて最適化されたアップデートを定期的にリリースしています。
  </Step>
  <Step num="2" title="Windows ゲームモードの設定">
    Windowsの設定を開き、「ゲーム」に移動して「ゲームモード」を「オン」に切り替えます。これにより、システムリソースがゲームクライアントに優先的に割り当てられます。
  </Step>
  <Step num="3" title="仮想メモリの割り当て調整">
    RAMが8GBの場合、SSD上のページファイルサイズを手動で増やすことで、激しいレイド中のメモリ不足によるクラッシュを防ぐことができます。
  </Step>
  <Step num="4" title="ゲーム内設定の最適化">
    ゲームを起動し、影の品質、反射の詳細、アンチエイリアスを下げます。これらのオプションは、VRAMとGPUの処理能力を最も多く消費します。
  </Step>
</Steps>

初めてゲームを起動する前に、このチェックリストを使用してすべての前提条件が満たされているか確認してください。

  id="pre-launch-checklist"

これらの手順に従うことで、低スペックのシステムでも平均フレームレートを最大20%向上させることができます。
</Callout>
