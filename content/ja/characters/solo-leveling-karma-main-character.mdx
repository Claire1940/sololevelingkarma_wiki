<Section icon={Map} title="『俺だけレベルアップな件：KARMA』主人公：水篠旬とは？">

Netmarble Neoが開発する新作ローグライト・アクションRPGでは、プレイヤーは究極の「影の君主」となって戦いに身を投じます。**『俺だけレベルアップな件：KARMA』の主人公**は他でもない**水篠旬**ですが、低ランクダンジョンで苦戦していたお馴染みの主人公ではありません。本作では、全盛期を迎えた絶対最強の状態の水篠旬が登場します。

ストーリーは、原作シリーズのタイムラインにおける未探索の空白期間を埋めるものです。輪廻の杯を使用して時間を巻き戻した後、旬は君主たちとの孤独で過酷な戦いに挑むため、**次元の狭間**へと入ります。現実世界ではわずか2年しか経過していないものの、旬は人類の未来を守るために27年間に及ぶ壮絶な戦いに耐え抜きます。このオリジナルストーリーの核心となるのが**壊れた輪廻の杯**であり、プレイヤーは旬の断片化された記憶を取り戻すことで、このアーティファクトを修復しなければなりません。

| キャラクターの特徴 | 詳細仕様 | ゲーム内のステータス |
| :--- | :--- | :--- |
| **主人公** | 水篠旬（影の君主） | プレイ可能 |
| **声優** | Aleks Le（英語音声キャスト） | 確定済み |
| **世界観設定** | 次元の狭間（27年間に及ぶ戦争） | メインキャンペーン |
| **主な目的** | 壊れた輪廻の杯の修復 | コアミッション |
| **敵対者** | アンタレス（破滅の君主 - CV: Troy Baker） | 主要ボス |

<Section icon={Swords} title="戦闘メカニクスと武器の切り替え">

本作の戦闘は、固定されたクラス（職業）システムから脱却し、非常にダイナミックでテンポの速いクォータービューのハックアンドスラッシュシステムを採用しています。**『俺だけレベルアップな件：KARMA』の主人公**である水篠旬は、単一の武器種に縛られません。プレイヤーは戦闘中に複数の武器をシームレスに切り替えて、変化する敵の編成やボスのフェーズに対応することができます。

各武器タイプには、独自のステータス補正、攻撃速度、戦術的な利点があります：

<Section icon={Sparkles} title="祝福システムとローグライト進行">

君主の軍勢が繰り出す終わりのない波を生き抜くためには、**「祝福システム」**をマスターする必要があります。本作はローグライト構造を採用しているため、ダンジョンに挑むたびに初期状態からスタートし、動的に進化していきます。敵を倒してエリアをクリアするごとに、旬のスキルやステータスを変化させるランダムな「祝福」を選択します。

これらの祝福により、状況に応じて特化したビルドをその場で構築できるため、毎回異なるプレイ体験を楽しむことができます。

<Steps>
  <Step num="1" title="ベース武器の選択">
    挑戦を開始する前に最大3つの武器種を装備し、基本的な攻撃パターンを決定します。
  </Step>
  <Step num="2" title="断片化された記憶の収集">
    次元の狭間でエリートモンスターを倒して記憶の欠片をドロップさせ、祝福の選択肢をアンロックします。
  </Step>
  <Step num="3" title="祝福の相乗効果（シナジー）">
    ステータスを強化する祝福と属性効果（雷、氷結、岩石）を組み合わせて、武器スキルを増幅させます。
  </Step>
  <Step num="4" title="シャドウコンボの発動">
    召喚した影の兵士による自動追撃をトリガーする祝福を装備します。
  </Step>
</Steps>

| 祝福のカテゴリ | 一般的な効果 | プレイスタイルのシナジー |
| :--- | :--- | :--- |
| **攻撃的な祝福** | クリティカル率の上昇、武器への属性付与 | ハイリスクなガラスの砲台ビルド |
| **防御的な祝福** | HP自動回復、回避時のシールド、防御力スケーリング | 安全で維持力を重視した攻略 |
| **影の祝福** | 召喚クールダウンの短縮、影のダメージ強化 | 召喚師に特化した戦場支配 |

<Callout type="info" title="ビルドの最適化">
氷結属性を付与したスキルと高速な短剣攻撃を組み合わせることで、ボスをその場に凍りつかせ、背後からバックスタブを叩き込んで莫大なクリティカルダメージ倍率を叩き出すことができます。
</Callout>

</Section>

<Section icon={Shield} title="伝説の影の軍団を率いる">

『俺だけレベルアップな件』のゲームを語る上で、象徴的な**「影の軍団」**は欠かせません。至高の影の君主として、水篠旬は倒した敵から蘇らせた忠実なアンデッド兵士の軍団を率います。戦闘中、これらの影は召喚可能な味方としてプレイヤーと共に戦い、敵のヘイトを引きつけたり、派手なコンボスキルを発動したりします。

確認されている召喚メンバーの中には、以下の伝説的な将軍たちを呼び出すことができます：

- **イグリット**: 旬のアクティブスキルと連携して、素早く壊滅的な剣コンボを繰り出す忠実な赤き騎士の指揮官。
- **アイアン**: 周囲 of 敵を挑発し、強力な一撃を吸収して遠距離プレイスタイル用の安全地帯を作り出す、巨大な盾を持つタンク。

| 影の味方 | 戦場での役割 | 究極アビリティ |
| :--- | :--- | :--- |
| **イグリット** | 近接DPS / コンボパートナー | 同調したクロススラッシュの実行 |
| **アイアン** | タンク / クラウドコントロール | 挑発とよろめきを誘発する巨大な叩きつけ |

<Callout type="success" title="影のヒント">
ボスがよろめいているフェーズ（スタッガー状態）を狙って召喚を行いましょう。ボスがダウンしている時にイグリットを召喚すると、独自のデュアルストライク演出が発動し、大ダメージを与えることができます。
</Callout>

</Section>
