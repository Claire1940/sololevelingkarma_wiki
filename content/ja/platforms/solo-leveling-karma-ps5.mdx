コンソール版には、激しい戦闘シーンでパフォーマンスを維持するための動的解像度スケーリング機能が搭載されています。複数のハンターが同時にアルティメットスキルを発動すると、エンジンが一時的に内部解像度を調整し、スタッター（カクつき）を防ぎます。

<Section icon={Shield} title="コントローラーレイアウトとDualSense設定">

戦闘システムでは、コンボ、回避行動、クイックタイムイベントの正確な実行が求められます。コントローラーのレイアウトをカスタマイズすることで、反応速度を大幅に向上させることができます。デフォルトの操作設定では主要な攻撃がフェイスボタンに割り当てられていますが、戦闘中にカメラをより良く操作するために、回避やスキルをバンパーボタンに割り当てることを好むプレイヤーもいます。

DualSenseコントローラーは、武器の重さに対応した没入感のあるハプティックフィードバックを提供します。重い大剣は力強くゆっくりとした振動を、素早い短剣は軽く速いパルスを生成します。

| アクション | デフォルトボタン | 推奨代替ボタン | 変更の目的 |
| :--- | :--- | :--- | :--- |
| **通常攻撃** | □ | □ | 標準的なコンボ実行 |
| **回避 / ダッシュ** | ○ | R1 | 右スティックから親指を離さないため |
| **ハンタースキル1** | △ | △ | 主要スキルへの素早いアクセス |
| **ハンタースキル2** | × | × | 二次スキルへの素早いアクセス |
| **極限回避** | R1 | L1 | カウンターのタイミングを合わせやすくする |

<Steps>
  <Step num="1" title="設定メニューにアクセス">
    メインメニューから歯車アイコンを選択し、コントローラーのサブメニューを開きます。
  </Step>
  <Step num="2" title="カスタムプリセットを選択">
    操作設定を「デフォルト」から「カスタム」に切り替え、個別のボタンマッピングをアンロックします。
  </Step>
  <Step num="3" title="回避操作を再バインド">
    回避機能をバンパーボタンに割り当て、カメラ回転と移動を同時に行えるようにします。
  </Step>
  <Step num="4" title="トリガー感度の調整">
    トリガーのデッドゾーンを低く設定し、トリガーに割り当てたスキルが即座に発動するようにします。
  </Step>
</Steps>

<Callout type="warning" title="トリガーの疲労">
アダプティブトリガーはスキル発動時に抵抗を加えます。長時間プレイして手が疲れる場合は、ゲームオプションでトリガーエフェクトの強度を下げることを検討してください。
</Callout>

<Section icon={Target} title="トラブルシューティングとシステムメンテナンス">

コンソール版向けに最適化されていても、時折パフォーマンスの低下、ネットワーク切断、視覚的な不具合が発生することがあります。ゲームクライアントを最新の状態に保ち、十分なストレージ容量を確保することで、一般的なエラーを防ぐことができます。本作は進行状況の記録にアクティブなインターネット接続が必要となるため、安定したネットワーク環境を強く推奨します。

フレームレートの低下が続く場合は、コンソールのシステムキャッシュをクリアするか、ゲームデータを再インストールすることで、基盤となるファイルの破損問題を解決できる場合があります。

| 問題 | 考えられる原因 | 推奨される解決策 |
| :--- | :--- | :--- |
| **ロビーでのカクつき** | 人口過密 | 人の少ないチャンネルに移動する |
| **接続エラー** | 不安定な無線信号 | 可能であれば有線LAN接続を使用する |
| **テクスチャの表示遅延** | 外付けドライブのボトルネック | ゲームをコンソール内蔵のSSDに移動する |
| **HDRが白っぽく見える** | 不適切なシステム調整 | システムダッシュボードでHDR設定を調整する |

<Callout type="warning" title="ストレージ容量の必要性">
一時的なパッチファイルやシステムアップデートに対応するため、インストールサイズに加えて少なくとも10 GBの空き容量を確保してください。
</Callout>
