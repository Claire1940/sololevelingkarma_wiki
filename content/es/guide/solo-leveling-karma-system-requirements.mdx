<Section icon={Shield} title="Requisitos del sistema de Solo Leveling Karma: Especificaciones para PC">

Ejecutar el juego en PC ofrece la experiencia visual más impresionante, con resolución de sombras mejorada, iluminación global y tasas de fotogramas ilimitadas. Los jugadores que busquen el máximo rendimiento deben revisar los niveles de hardware a continuación para asegurarse de que su equipo esté listo para la acción.

El nivel mínimo tiene como objetivo ofrecer una resolución de 1080p a 30 fotogramas por segundo en ajustes bajos, mientras que el nivel recomendado apunta a 1080p a 60 fotogramas por segundo en ajustes altos.

| Componente | Especificación mínima | Especificación recomendada |
|---|---|---|
| SO | Windows 10 (64 bits) | Windows 10 / 11 (64 bits) |
| CPU | Intel Core i5-8400 / AMD Ryzen 5 1600 | Intel Core i7-9700K / AMD Ryzen 7 3700X |
| RAM | **8 GB RAM** | **16 GB RAM** |
| GPU | NVIDIA GTX 1060 6GB / AMD RX 580 | NVIDIA RTX 2060 / AMD RX 5600 XT |
| DirectX | Versión 11 | Versión 12 |
| Almacenamiento | 50 GB de espacio (HDD) | 50 GB de espacio (SSD) |

Para los jugadores entusiastas que desean experimentar el juego en resolución 4K con ajustes al máximo y trazado de rayos activado, es necesaria una configuración de nivel elite. Esta configuración aprovecha tecnologías avanzadas de escalado como NVIDIA DLSS o AMD FSR para mantener tasas de fotogramas estables durante intensas batallas contra jefes.

| Componente | Especificación Ultra 4K |
|---|---|
| SO | Windows 11 (64 bits) |
| CPU | Intel Core i7-12700K / AMD Ryzen 7 5800X3D |
| RAM | **32 GB RAM** |
| GPU | NVIDIA RTX 4070 / AMD RX 7800 XT |
| DirectX | Versión 12 |
| Almacenamiento | 50 GB de espacio (SSD NVMe) |

<Callout type="warning" title="Aviso de almacenamiento">
Instalar el juego en una unidad de disco duro estándar (HDD) puede provocar desincronización de audio, carga lenta de texturas y transiciones prolongadas en las pantallas de carga.
</Callout>

<Section icon={Swords} title="Requisitos del sistema móvil y rendimiento">

Netmarble ha optimizado la versión móvil para que funcione en una amplia gama de dispositivos. Sin embargo, debido a la naturaleza trepidante del sistema de combate, contar con un dispositivo que cumpla con las especificaciones recomendadas es crucial para evitar el estrangulamiento térmico (thermal throttling) y caídas repentinas de fotogramas durante peleas concurridas contra jefes.

|---|---|---|
| SO Android | Android 9.0 o superior | Android 11.0 o superior |

| RAM Android | **4 GB RAM** | **8 GB RAM** |
| SO iOS | iOS 14 o superior | iOS 16 o superior |
| Dispositivo iOS | iPhone XS / iPad Air 3 | iPhone 14 Pro / iPad Pro (M1) |
| RAM iOS | **4 GB RAM** | **6 GB RAM** |

<Section icon={Sparkles} title="Guía de optimización paso a paso">

Si tu sistema apenas cumple con los requisitos mínimos, puedes realizar varias optimizaciones de software para exprimir un rendimiento adicional de tu equipo. Sigue estos pasos para preparar tu PC para el juego.

<Steps>
  <Step num="1" title="Actualizar los controladores gráficos">
    Descarga los controladores más recientes directamente de NVIDIA, AMD o Intel. Los fabricantes de GPU lanzan regularmente actualizaciones optimizadas específicamente para los nuevos lanzamientos de Netmarble.
  </Step>
  <Step num="2" title="Configurar el Modo de juego de Windows">
    Abre la configuración de Windows, navega a Juegos y activa el "Modo de juego". Esto prioriza los recursos del sistema para el cliente del juego.
  </Step>
  <Step num="3" title="Ajustar la asignación de memoria virtual">
    Si tienes 8 GB de RAM, aumenta manualmente el tamaño del archivo de paginación en tu SSD para evitar bloqueos por falta de memoria durante las incursiones intensas.
  </Step>
  <Step num="4" title="Optimizar los ajustes del juego">
    Inicia el juego y reduce la calidad de las sombras, los detalles de los reflejos y el suavizado de contornos (anti-aliasing). Estas opciones consumen la mayor cantidad de VRAM y potencia de procesamiento de la GPU.
  </Step>
</Steps>
